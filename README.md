# 🕵️ Projeto Detective Quest – Investigação em uma Mansão

Simulação de um **jogo investigativo**, onde o jogador explora uma mansão para encontrar pistas e descobrir o culpado.

## 🎯 Objetivo

Coletar pistas suficientes durante a exploração da mansão para identificar o suspeito correto.

## 🧩 Estruturas utilizadas

* 🌳 **Árvore Binária**
  Representa os cômodos da mansão.

* 🌲 **Árvore BST**
  Armazena as pistas encontradas.

* #️⃣ **Tabela Hash**
  Relaciona pistas aos suspeitos.

## 🔍 Funcionalidades do sistema

* Explorar cômodos
* Coletar pistas
* Listar evidências
* Associar pistas a suspeitos
* Identificar o culpado

# 🎮 Projeto Free Fire – Sistema com Algoritmos de Ordenação

Projeto inspirado em gerenciamento de itens dentro de um sistema semelhante a inventário de jogo.

## 🎯 Objetivo

Criar um sistema que organize e busque itens utilizando diferentes algoritmos.

## ⚙️ Algoritmos implementados

* 🔁 **Bubble Sort**
* 🔁 **Insertion Sort**
* 🔁 **Selection Sort**
* 🔍 **Busca Binária**

## 📦 Estrutura do item

Cada item possui:

* Nome
* Tipo
* Quantidade
* Prioridade

## 🔧 Funcionalidades

* Inserir item
* Remover item
* Ordenar inventário
* Buscar itens rapidamente
* Exibir lista de itens

# 📚 Projeto Tetris Stack – Pilha e Fila

Projeto que implementa duas estruturas fundamentais da computação.

## 🧱 Estruturas utilizadas

### 📥 Fila (Queue)

Sistema **FIFO – First In First Out**

Operações:

* Inserir elemento
* Remover elemento
* Exibir fila

### 📤 Pilha (Stack)

Sistema **LIFO – Last In First Out**

Operações:

* Empilhar elemento
* Desempilhar elemento
* Visualizar topo da pilha

# ⚙️ Compilação e Execução

Para executar os programas utilize o compilador **GCC**:

```bash
gcc nome_do_programa.c -o programa
./programa
```

Exemplo:

```bash
gcc DetectiveQuest_Fase3.c -o detective
./detective
```
